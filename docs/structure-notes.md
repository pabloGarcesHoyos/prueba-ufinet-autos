# Structure Notes

## Root

- `backend/`: Spring Boot API base structure.
- `frontend/`: React application base structure.
- `database/`: SQL scripts and future schema assets.
- `docs/`: Supporting technical documentation and API tooling files.

## Backend

- `src/main/java/com/ufinet/autos/config`: Shared configuration placeholders.
- `src/main/java/com/ufinet/autos/controller`: REST controller placeholders.
- `src/main/java/com/ufinet/autos/dto`: Request and response models to be defined later.
- `src/main/java/com/ufinet/autos/entity`: Persistence model placeholders.
- `src/main/java/com/ufinet/autos/exception`: Centralized error handling placeholders.
- `src/main/java/com/ufinet/autos/repository`: Persistence contract placeholders.
- `src/main/java/com/ufinet/autos/security`: Security and JWT placeholders.
- `src/main/java/com/ufinet/autos/service`: Service contracts and implementations.
- `src/main/resources`: Base runtime configuration.

## Frontend

- `public/`: Static public assets.
- `src/api`: HTTP client setup placeholder.
- `src/assets`: Internal static assets.
- `src/components`: Reusable UI building blocks.
- `src/context`: Shared React context placeholders.
- `src/hooks`: Custom hooks placeholders.
- `src/pages`: Page-level views for auth and cars.
- `src/routes`: Application routing structure.
- `src/services`: API-facing service placeholders.
- `src/styles`: Shared styling and design tokens.
- `src/utils`: Shared constants and helpers.
